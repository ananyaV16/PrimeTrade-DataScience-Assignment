## Project Overview

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data from Hyperliquid.

The objective is to determine whether market sentiment influences trader profitability, leverage usage, trade size, and trading behavior, and to derive insights that can support better trading strategies.

---

## Datasets

### 1. Bitcoin Fear & Greed Index
- Date
- Market Sentiment Classification (Fear / Greed)

### 2. Hyperliquid Historical Trader Data
Includes:
- Account
- Symbol
- Execution Price
- Trade Size
- Side (Buy/Sell)
- Leverage
- Closed PnL
- Timestamp
- Event
- Start Position
